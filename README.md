# 🎥 Enterprise Zoom Clone

A modern **Enterprise Video Conferencing Platform** built with **Next.js, TypeScript, Tailwind CSS, Clerk Authentication, Stream Video SDK, and Shadcn UI**. The application enables secure video meetings, screen sharing, meeting scheduling, recordings, and collaborative communication with an intuitive user interface.

---

# 🚀 Features

* 🔐 Secure User Authentication with Clerk
* 🎥 High-Quality Video & Audio Calls
* 👥 Instant Meeting Creation
* 📅 Schedule Future Meetings
* 🔗 Share Meeting Links
* 📺 Screen Sharing
* 🎙️ Microphone & Camera Controls
* 💬 In-Meeting Chat Support
* 📹 Meeting Recordings
* 📋 Personal Meeting Dashboard
* 🌙 Responsive & Modern UI
* 📱 Mobile Friendly Design

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/your-username/enterprise-zoom-clone.git

cd enterprise-zoom-clone
```

---

## Install Dependencies

```bash
npm install
```

---

## Configure Environment Variables

Create a `.env.local` file.

```env
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=your_clerk_publishable_key
CLERK_SECRET_KEY=your_clerk_secret_key

NEXT_PUBLIC_STREAM_API_KEY=your_stream_api_key
STREAM_SECRET_KEY=your_stream_secret_key
```

---

## Run Development Server

```bash
npm run dev
```

Application will be available at:

```
http://localhost:3000
```
